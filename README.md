# @amlplugins/google-cloud-translate

Native `@google-cloud/translate` SDK re-exported under the `@amlplugins` namespace.

> Cloud Translation.

## Install

```bash
npm install @amlplugins/google-cloud-translate
```

## Use

```ts
import { /* named exports from @google-cloud/translate */ } from "@amlplugins/google-cloud-translate";
// or
import sdk from "@amlplugins/google-cloud-translate";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
